# Meeting Transcript

Team meeting transcript app built with Next.js, Neon Auth, Neon Postgres,
Cloudflare R2, Recall.ai, ElevenLabs, and Inngest.

## Development

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Verification

```bash
npm run lint
npm test
npm run build
npm run test:e2e
```

## Environment

Copy `.env.example` to `.env.local` and fill in the required service values.
